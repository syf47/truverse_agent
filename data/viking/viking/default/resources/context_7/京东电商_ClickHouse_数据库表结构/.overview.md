## Detailed Description
### 1_jd_shopId_wares_京东商品数据表.md 1_jd_shopId_wares_京东商品数据表.md
This is a data table reference document for the `jd_{shopId}_wares` JD commodity data table, intended to define the table structure and field specifications of JD store commodity data. The document covers core basic attributes, price and inventory, commodity status, commodity details, logistics attributes, extended attributes, OCR results, and time fields. Key information includes the table uses MergeTree engine, is partitioned by createTime month, sorted by wareId and updateTime, with wareId as the primary key, and detailed field definitions and value descriptions such as commodity status and cooperation mode. It is suitable for data analysts, operation personnel and development staff to query and use commodity data, and supports business scenarios such as commodity information analysis, inventory management and operation status statistics of JD stores. Keywords: JD commodity data table, MergeTree, wareId, commodity status, price inventory, commodity attributes, OCR results.

### 2_jd_shopId_comments_京东商品评_2more.md 2_jd_shopId_comments_京东商品评_2more.md
This is a data table reference document that defines the structure of two JD e-commerce business data tables for data developers and analysts. The document is divided into two core parts, covering the `jd_{shopId}_comments` product review table and `jd_{shopId}_qa` user Q&A table respectively. For each table, it specifies storage attributes including engine, partition rule, sort key and TTL, and details the field name, data type and business meaning of different modules respectively. The review table includes core review, image, interaction and user information modules; the Q&A table stores one answer per row with redundant question information, including question and answer related fields. The document is mainly used for data query, ETL development and business analysis related to JD product reviews and user Q&A, requiring basic knowledge of ClickHouse table structure and e-commerce business. Key keywords include JD product review, user Q&A, MergeTree, sku_id, comment_id, question_id.

### 京东电商_ClickHouse_数据库表结构_2more.md 京东电商_ClickHouse_数据库表结构_2more.md
This reference document is designed to clarify the table structure and association rules of JD e-commerce ClickHouse databases for relevant developers and data analysts. The core content focuses on the association relationships between three business tables under the same store, namely the product table, comment table and Q&A table. Key points include: first, the shop ID placeholder in table names needs to be replaced with actual values during use; second, the product table associates with the comment table via product ID, and associates with the Q&A table via the string-formatted product ID; third, the comment and Q&A tables are also associated through product ID; fourth, each row of the Q&A table stores one answer, and question information is stored redundantly. This document is suitable for personnel engaged in JD e-commerce data query, statistics and business analysis work, who need to have basic knowledge of ClickHouse SQL and e-commerce business data. Keywords: JD e-commerce, ClickHouse, table structure, table association, product table, comment table, Q&A table, shop ID.
